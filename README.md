# Personal Cloud Infrastructure ☁️

Автоматизированное развертывание личного облачного хранилища на базе **Nextcloud** и СУБД **MariaDB** с использованием подхода Infrastructure as Code (IaC).

## 🛠 Стек технологий
* **Виртуализация:** Docker, Docker Compose
* **Сервисы:** Nextcloud (Apache + PHP), MariaDB 10.6
* **ОС:** Linux (Ubuntu/Mint)

## ⚙️ Особенности архитектуры
* Полная изоляция сервисов во внутренней сети Docker.
* Настроено персистентное хранение данных (Volumes) — база данных и пользовательские файлы монтируются на хост-машину, что гарантирует сохранность данных при пересоздании контейнеров.
* Настроен автозапуск сервисов (`restart: always`) при перезагрузке хост-машины.
* Игнорирование системных директорий баз данных через `.gitignore` для безопасности.

## 🚀 Как запустить (Deploy)
1. Клонировать репозиторий:
   `git clone https://github.com/a-berdaly/nextcloud-docker-deployment.git`
2. Перейти в директорию:
   `cd nextcloud-docker-deployment`
3. Поднять инфраструктуру в фоновом режиме:
   `docker compose up -d`
4. Облако будет доступно на порту `8080`.
